"# clearnote" 

Readme written using ClearNote

Clearnote was created as a replacement to the bloated notepad of new and avoid overloaded features of other notepads. This should be fine running on Mac OS, Windows and Linux. Builds are made available under releases to download. Or simple have Python installed and download the .py file.

Windows and Mac will show a message about unknown source, as no certificate for Mac or untrusted in Windows. They are built as is, but should work fine if you know how to ignore the messages and run it.
